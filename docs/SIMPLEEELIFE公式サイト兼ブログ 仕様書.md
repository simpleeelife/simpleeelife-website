## プロジェクト概要

SIMPLEEELIFE（シンプリィライフ）の公式サイトとブログを統合したWebサイトを構築する。「こんがらがった人生をシンプルな線にする知のレシピ」をコンセプトに、サイバーパンク風デザインとモダンな機能を融合したサイトを実現する。

## 技術スタック

フロントエンド: [Next.js](http://next.js/) (React + TypeScript)

スタイリング: Tailwind CSS + カスタムCSS（ネオン効果等）

CMS: Sanity

ホスティング: Vercel

分析: Google Analytics
